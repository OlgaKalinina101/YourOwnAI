# 🚀 Quick Start - View-Only Web Client

## 📱 На телефоне (Android)

```
1. Settings → Local Network Sync
2. Start Server ✅
3. Show QR Code 📱
```

## 💻 На компьютере

```
1. Scan QR Code 📷
   ИЛИ
   Открой http://<IP>:8765 в браузере

2. Готово! 🎉
```

---

## 🎨 Что можно делать

- ✅ **Смотреть** все чаты
- ✅ **Искать** по названиям  
- ✅ **Экспортировать** в MD/TXT
- ✅ **Копировать** сообщения
- ✅ **Просматривать** воспоминания

**❌ Писать сообщения НЕЛЬЗЯ** (только просмотр!)

---

## 🔧 Разработка (опционально)

```bash
cd web-client
npm install
npm start
```

Откроется `http://localhost:3000`

---

## ❓ Не работает?

**Connection Failed?**
- ✅ Оба устройства в одной Wi-Fi
- ✅ Server запущен на телефоне
- ✅ Попробуй QR код
- ✅ Отключи VPN

**Подробнее:** см. `VIEW_ONLY_SETUP.md`
